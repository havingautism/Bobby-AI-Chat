# 🐱 Bobby

一个现代化的AI聊天应用，基于React构建，支持多种AI模型，提供优雅的用户体验

## ✨ 功能特性

### 🎯 核心功能
- 📱 **完美响应式设计** - 无缝适配PC端和移动端，支持触摸操作
- 💾 **本地数据存储** - 聊天历史自动保存在浏览器本地存储，保护隐私
- 🔄 **多对话管理** - 支持创建、切换、删除多个对话会话
- ⚡ **实时打字动画** - 消息发送时显示优雅的打字指示器
- 📝 **完整Markdown支持** - AI回复支持代码高亮、表格、列表等格式
- 🔌 **硅基流动API** - 专注于硅基流动API服务，提供丰富的模型选择

### 🆕 高级功能
- 🔍 **智能搜索** - 搜索历史对话内容，支持关键词高亮显示
- 🎛️ **侧边栏控制** - PC端支持展开/收起侧边栏，节省屏幕空间
- ⚙️ **可视化设置** - 图形化API配置界面，支持连接测试
- 🎨 **主题系统** - 支持明亮模式和暗色模式切换
- 🌐 **多语言支持** - 支持中文和英文界面切换
- 📋 **消息操作** - 支持复制消息内容和重新生成回复

### 🎨 UI/UX 特性
- **现代化设计** - 采用最新的UI设计趋势，界面简洁美观
- **自动展开输入框** - 输入框根据内容自动调整高度
- **统一头像系统** - 所有头像保持一致的样式和大小
- **智能布局** - 移动端优化布局，充分利用屏幕空间
- **流畅动画** - 平滑的过渡动画和交互反馈

## 快速开始

### 1. 安装依赖

```bash
pnpm install
```

### 2. 配置API

首次使用时，应用会自动提示配置API：

1. 启动应用后，点击右上角的⚙️设置按钮
2. 系统默认配置硅基流动API服务
3. 输入API密钥
4. 选择合适的AI模型
5. 点击"测试连接"验证配置
6. 保存设置开始使用

### 3. 启动应用

```bash
pnpm start
```

应用将在 http://localhost:3000 启动。

## 🔧 API配置说明

### 硅基流动API (默认)

项目专注于硅基流动API服务，提供丰富的模型选择：

#### 支持的模型类型
- **最新模型** - DeepSeek-V3.1、Step-3等最新对话模型
- **通义千问3系列** - Qwen3系列模型，包括编程专用模型
- **推理模型** - DeepSeek-R1等高级推理模型
- **对话模型** - 各种对话优化模型
- **通义千问2.5系列** - Qwen2.5系列模型

#### 配置示例
```javascript
{
  baseURL: 'https://api.siliconflow.cn/v1/chat/completions',
  apiKey: 'YOUR_SILICONFLOW_API_KEY',
  model: 'deepseek-ai/DeepSeek-V3.1',
  temperature: 0.7,
  maxTokens: 2000
}
```

### 推荐模型
- **DeepSeek-V3.1** - 最新对话模型，性能优秀
- **DeepSeek-R1** - 高级推理模型，适合复杂问题
- **Qwen3系列** - 编程专用模型，代码生成能力强
- **Pro版本** - 性能更强，适合专业用途

## 📱 使用指南

### 基本操作
- **新建对话**: 点击"新建聊天"按钮创建新会话
- **发送消息**: 输入内容后按Enter或点击发送按钮
- **切换对话**: 点击侧边栏中的对话项
- **删除对话**: 悬停对话项，点击删除按钮
- **复制消息**: 点击消息下方的复制按钮
- **重新生成**: 点击重新生成按钮获取新的回复

### 高级功能

#### 🔍 智能搜索
- 在搜索框输入关键词查找历史对话
- 支持搜索对话标题和消息内容
- 匹配的关键词会高亮显示
- 显示搜索结果数量统计

#### 📂 侧边栏管理
- **展开状态**: 显示完整对话列表和搜索功能
- **收起状态**: 只显示图标，节省屏幕空间（仅PC端）
- **智能分组**: 按"今天"和"之前"自动分组
- **响应式**: 移动端自动适配

#### ⚙️ 设置面板
- **API配置**: 可视化配置界面，专注于硅基流动
- **模型选择**: 丰富的模型列表，按类型分类
- **连接测试**: 一键测试API连接状态
- **参数调节**: 调整温度、最大令牌数等参数

#### 🎨 主题系统
- **明亮模式**: 适合白天使用，界面清爽
- **暗色模式**: 适合夜间使用，护眼舒适
- **自动切换**: 根据系统主题自动切换

#### 🌐 多语言支持
- **中文界面**: 默认中文界面
- **英文界面**: 支持英文界面切换
- **实时切换**: 无需刷新页面即可切换语言

### 移动端适配
- 在移动设备上，侧边栏会自动隐藏
- 点击左上角的☰按钮可打开/关闭侧边栏
- 触摸友好的界面设计
- 完美适配各种屏幕尺寸
- 移动端优化布局，头像和文字垂直排列

### 数据存储
- 所有聊天记录自动保存在浏览器的localStorage中
- API配置和用户设置同样本地保存
- 刷新页面或重新打开应用时会自动加载历史记录
- 数据完全存储在本地，保护隐私安全

## 自定义配置

### 修改主题颜色
编辑 `src/styles/theme.css` 文件中的CSS变量：
```css
:root {
  --accent-color: #10a37f;
  --bg-primary: #ffffff;
  --text-primary: #1a1a1a;
  /* 更多颜色变量... */
}
```

### 添加新功能
项目采用模块化设计，可以轻松添加新功能：
- 在 `components/` 目录下添加新组件
- 在 `utils/` 目录下添加工具函数
- 修改 `App.js` 集成新功能

## 技术栈

- **前端框架**: React 18
- **样式**: CSS3 + Tailwind CSS
- **状态管理**: React Hooks
- **API调用**: Fetch API
- **本地存储**: localStorage
- **构建工具**: Vite

## 注意事项

1. **API密钥安全**：请不要将API密钥提交到公共代码仓库
2. **浏览器兼容性**：建议使用现代浏览器（Chrome、Firefox、Safari、Edge）
3. **存储限制**：localStorage有存储大小限制，大量对话可能需要清理
4. **网络要求**：需要稳定的网络连接访问硅基流动API
